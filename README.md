## **🧾기능설명**
- 시스템에 회원가입을 하고, 동아리에 가입, 탈퇴, 정보조회를 할 수 있다.

## 🧑‍🤝‍🧑사용자 기능
> ❓아이디가 필요한 모든 작업은 로그인 이후에 가능하게 설정해놓음❓
1. 회원가입
2. 로그인 & 로그아웃
3. 회원 조회 : index, userId 두 가지 방법으로 조회 가능
4. 회원 탈퇴
5. 회원 전체 조회
6. 회원 정보 수정

## 🙆동아리 기능
1. 동아리 리스트 확인
2. 동아리 상세 보기
3. 동아리 가입 하기
4. 동아리 탈퇴 하기
